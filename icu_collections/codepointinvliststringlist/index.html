Module icu_collections::codepointinvliststringlist

source ·
Expand description

This module provides functionality for querying of sets of Unicode code points and strings.

It depends on CodePointInversionList to efficiently represent Unicode code points, while it also maintains a list of strings in the set.

It is an implementation of the existing ICU4C UnicodeSet API.
